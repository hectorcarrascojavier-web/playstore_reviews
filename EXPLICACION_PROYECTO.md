# PlayStore Reviews - Mini Proyecto Django

## Nombre del proyecto
PlayStore Reviews

## Problema que resuelve
Permite a los usuarios registrar y visualizar reseñas de aplicaciones Android para tomar mejores decisiones antes de descargar una app.

## Modelo creado y sus campos
Modelo: AppReview
- nombre_app: nombre de la aplicación
- categoria: categoría (Juegos, Social, Productividad, etc.)
- calificacion: puntuación de 0 a 5 estrellas
- descargas: rango de descargas (ej. "100M+")
- reseña: comentario del usuario
- fecha_registro: fecha automática

## Funcionalidades
- Página principal con descripción del proyecto
- Página de listado con tarjetas de reseñas
- Panel administrativo para gestionar registros
- Diseño responsive con Tailwind CSS
